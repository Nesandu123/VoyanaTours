# Voyana Tours - React One Page Tourism Website

A modern one-page tourism landing page built with **React + Vite + Framer Motion**.

## Included Sections
- Navbar
- Hero section with staggered text animation and parallax background
- Services section with scroll reveal cards
- Tour packages section
- Tour guides and transport section
- About / travel operator section
- Contact section with animated inquiry form
- Footer

## Animations Included
- Navbar slide/fade on load
- Staggered hero text entrance
- Floating background blobs
- Hover and tap button motion
- Scroll reveal on cards and sections
- Fade-up contact area
- Smooth section transitions
- Hero parallax effect
- Reduced motion support for accessibility

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
- Replace placeholder text, email, phone, and package data with your real tourism business details.
- The contact form is front-end only and ready to connect to your backend, WhatsApp, Formspree, EmailJS, or custom API.
